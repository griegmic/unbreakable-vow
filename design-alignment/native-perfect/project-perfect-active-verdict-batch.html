Donald Pre-Audit Notes

Parity: These states follow the web phases: witness pending, active, and verdict waiting.

Native Advantage: Joe accepted gets a brief success state with haptic success, then lands on active detail. The web can skip this, but native should acknowledge the push/event.

Truthfulness: Shared-link states use future-role language until someone accepts. The app never claims a text was delivered or that Joe already has the verdict link.

CTA Strategy: Active states use one primary action. Mid-vow lets the maker exit cleanly; near-deadline makes the final check-in primary; verdict due shifts to witness nudge.
