Recommendation

1. Enough instruction? Yes. “They keep you honest. They decide whether you kept it.” plus the Add a judge subtitle is enough for a first-time user because the ready card shows what they are asking someone to judge. More explanation would slow the moment down.

2. Add judge above ready to seal? I would not. Judge-first is faster, but it feels more like an address-book task. Ready-first reminds the user what is at stake, then asks who should hold them to it.

3. Should Ready to seal exist? Yes. It reduces anxiety before asking for contacts and reinforces the consequence. Keep it compact and do not let it push Add a judge below the fold.

Vote: Option A is the Project Perfect candidate. Option B is viable if device testing shows users hesitate because Add a judge is too low. Option C is too sparse and loses the vow/consequence context.
