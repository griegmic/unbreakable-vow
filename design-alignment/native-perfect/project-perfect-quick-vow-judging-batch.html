Quick Vow: This is intentionally the mobile-web power page, not a new native invention. The one native delta is contacts-first witness selection, because the app has the address book and the social step is high leverage.

Judging detail: The screen answers three questions in three seconds: who am I holding accountable, what did they vow, and when do I decide? The primary action while active is relational, not administrative: text them.

Verdict due: The CTA pair is deliberately blunt. At verdict time, the user does not need more education; they need to make the call. “Text Sarah first” stays quiet for empathy.

Witness accept/deny: This stays close to mobile web public witness flow. It does not assume the witness has the app. The app-native version can reuse this design if a future installed-app witness path exists.
