Cross-platform note: The Expo app's Quick Vow stays as-is (single page with native contact picker + haptics). The web-only two-step redesign is appropriate because web lacks those native affordances that make single-page density tolerable. The mental model is the same on both platforms: compose → confirm terms → seal. The implementation just adapts to each platform's strengths.
