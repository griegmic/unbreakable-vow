# Design and build a darker, more magical Unbreakable Vow app

Features

- [x] Create a vow in seconds from a striking, low-friction opening screen with one clear input and smart example prompts.
- [x] Refine the vow into sharper, more enforceable wording with elegant suggestions that feel helpful, not robotic.
- [x] Choose a witness through a premium invite flow that feels personal, social, and high-stakes.
- [x] Set the money at risk and choose the consequence in a way that feels clean, serious, and emotionally weighty.
- [x] Sign in with Apple, Google, or email in a polished auth step that feels trustworthy without slowing the flow down.
- [x] Review the full vow in a dramatic sealing step with a memorable ritual-like moment before it goes live.
- [x] Live the vow in a three-way SMS group thread with your witness and Oathkeeper, the AI that checks in during the week and prompts the verdict on day 7.
- [x] See post-seal states including waiting for acceptance, live vow with countdown, witness invite, vow kept, vow broken, and history.
- [x] View streaks, money protected, and outcomes so the app feels motivating and fintech-sharp.

Design

- [x] Use a dark, modern visual system inspired by premium fintech apps, with near-black surfaces, warm gold highlights, and restrained green/red verdict states.
- [x] Keep the magical influence subtle and elegant, with ritual energy shown through glow, sealing moments, depth, and typography rather than costume-like fantasy visuals.
- [x] Make the opening screen feel exceptional: atmospheric, editorial, and instantly understandable with no information overload.
- [x] Give cards and buttons a sculpted, high-end feel with layered shadows, soft borders, and selective glow instead of generic gradients.
- [x] Use motion to create delight in a few high-impact moments like vow analysis, sealing, and verdict reveals. Kept should feel like relief and pride. Broken should feel heavy and final.
- [x] Keep every screen mobile-first, spacious, and touch-friendly so it feels native rather than like a shrunk website.

Pages / Screens

- [x] Opening screen with the vow input, a strong headline, and a few beautifully presented vow examples.
- [x] Vow refinement screen that clarifies weak wording and helps the user lock in something measurable.
- [x] Witness screen that guides the user into choosing, naming, and inviting the right person.
- [x] Stakes screen that sets the amount and consequence with emotional clarity.
- [x] Sign-in screen with real authentication choices and a trustworthy tone.
- [x] Sealing screen that summarizes the vow and delivers the signature ritual moment.
- [x] Waiting / sent screen showing the vow has been delivered and what happens next.
- [x] Live vow screen showing the active status, timeline, and outcome at stake.
- [x] Witness invite preview screen showing the other side of the experience.
- [x] Witness verdict screen with two-tap confirmation so the decision feels irreversible and weighty.
- [x] Outcome screens for both vow kept and vow broken, each with strong emotional contrast. Broken includes a donation receipt.
- [x] History screen with record, streak, and past vow summaries.

Product direction

- [x] Make the witness mechanic feel central to the product's identity so the social accountability and virality come through immediately.
- [x] Preserve the strong core flow from the current concept, but tighten every screen so the path feels faster, clearer, and more premium.
- [x] Prioritize emotional clarity and speed over feature breadth so the experience feels obvious on first use.
